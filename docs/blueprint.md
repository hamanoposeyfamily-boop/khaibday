# **App Name**: TwoMuchFun

## Core Features:

- Bento Grid Dashboard: A high-performance interactive invitation UI utilizing a Material You-inspired bento layout for all event details.
- AI Excuse Neutralizer: An AI-powered tool that analyzes attendee hesitation and generates witty, sarcastic responses to common family event excuses.
- Persistent RSVP Hub: A streamlined confirmation system to track guest counts and integrate directly with your hosting environment.
- Live Tick Countdown: A real-time synchronization tool that visually displays the seconds remaining until the celebration starts.
- Haptic Block Interactions: Tactile animations and mobile vibration sequences paired with jiggling interactive toy blocks.
- Smart .ics Integration: Automatic generation and export of calendar events for seamless scheduling across Apple and Google Calendars.

## Style Guidelines:

- Primary color: Vibrant Tech Blue (#4285F4) chosen for its clean, modern energy and connection to familiar design systems.
- Background color: Sky Tint White (#F1F4F9), a deeply desaturated version of the primary blue for a fresh, spacious canvas.
- Accent color: Deep Indigo (#4D42F4) to provide sharp contrast and an analogous sophistication to navigation elements.
- Recommendation: Use 'Fredoka' for playful headers and 'Outfit' for modern, clean body text. Note: currently only Google Fonts are supported.
- Bento-style grid with significant border-radius (40px) and tonal surface divisions for a modern 'Material' feel.
- Playful floating transitions and ink-ripple effects on all actionable components to maintain tactile engagement.
